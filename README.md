Quick Rails
===========

**Repositório em construção**

Scripts e arquivos para configurar rapidamente um novo servidor pronto para rodar aplicações Rails.

O objetivo é configurar:

* Usuário
* Dependências
* Postgresql
* Nginx
* Rbenv
* Ruby
* Bundler
* Rails
* Upstart para Sidekiq
* Upstart para Puma
* Redis
